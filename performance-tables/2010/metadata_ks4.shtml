Data description page

This description file contains variables names and descriptions for use with the downloadable 2010 Secondary School (Key Stage 4) Performance Tables. There are 5 different record types identifiable by the first field RecType. Not all variables are populated for all record types. For instance, school details are not populated for record types 4, 5 and 7 which are Local and National averages. This datafile was created on mer. mars 26 19:25:28 GMT 2014 .
