Data description page

This description file contains variable names and descriptions for use with the downloadable 2010 Primary School Performance Tables. There are 5 different record types identifiable by the first field RecType. Not all variables are populated for all record types. For instance, school details are not populated for record types 3,4 and 5 which are Local and National averages. This page was created on mer. mars 26 19:25:21 GMT 2014 .
